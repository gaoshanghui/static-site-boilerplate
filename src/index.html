Static Site Boilerplate

Version: 0.1.0 beta

Quick Start

  1. Install by running "npm install" command.

  2. Run "npm start" to enter development mode.

  3. Run "npm build" to create a production build.

*To use the boilerplate you have to pre-install Node.js.

*This is a beta version of the static site boilerplate. Feel free to play with it.

*The goal is to build a boilerplate that has more features with high build performance.

*The boilerplate now support using React(beta 2021.07.12).
